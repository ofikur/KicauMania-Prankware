Kicaumania Prankware adalah sebuah perangkat lunak lelucon interaktif yang dirancang untuk sistem operasi Windows. Aplikasi ini berjalan di latar belakang dengan menampilkan animasi overlay transparan pada layar dan memutar audio secara berulang. 

Program ini dikembangkan murni untuk tujuan hiburan dan demonstrasi pemrograman antarmuka Windows API. Perangkat lunak ini diklasifikasikan sebagai aman, tidak mengubah registri inti, tidak mencuri data, dan tidak merusak fungsi sistem operasi.

## Fitur Utama
* **Click-Through Interface**: Layar animasi bersifat tembus pandang dan tembus klik. Pengguna tetap dapat mengoperasikan kursor, mengetik, dan membuka aplikasi lain di belakang overlay tanpa gangguan fungsionalitas.
* **Auto-Run Persistence**: Program dirancang untuk mereplikasi dirinya ke dalam direktori Startup Windows, sehingga efek lelucon akan tetap berjalan secara otomatis setiap kali sistem dihidupkan ulang.
* **Non-Destructive**: Seluruh skrip berjalan di level antarmuka pengguna (UI) dan sama sekali tidak melibatkan modifikasi sistem yang merusak.
* **Dedicated Uninstaller**: Repositori ini dilengkapi dengan skrip pembersih khusus untuk menghentikan proses dan menghapus jejak program dari sistem secara permanen dan aman.

## Panduan Penggunaan

### 1. Menjalankan Program
Jalankan file executable utama (`kicaumania.exe`). Program akan langsung aktif, menerapkan overlay pada layar, memutar audio, dan mendaftarkan dirinya ke dalam sistem startup untuk eksekusi otomatis pada sesi berikutnya.

### 2. Menghentikan dan Membersihkan Program
Untuk menghentikan animasi dan menghapus program dari sistem secara permanen, jangan menghapus file secara manual saat program sedang berjalan. Gunakan file pembersih yang telah disediakan:
1. Jalankan `uninstaller.exe`.
2. Program akan secara otomatis menghentikan proses latar belakang dan menghapus file dari direktori startup.
3. Kotak pesan sistem akan muncul mengonfirmasi bahwa sistem telah dibersihkan.
